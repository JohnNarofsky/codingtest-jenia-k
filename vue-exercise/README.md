# mstar-assignment

This test requires an install of a [node version](https://nodejs.org/en/download/package-manager) version of 20 or greater. 

## Project Setup

To get started, unzip this file into a development directory. Run the following commands in a bash terminal of your choice:

```sh
npm install
```

```sh
npm run dev
```

Once it's running, navigate to [the test site](http://localhost:5173/). Hotloading is enabled.
